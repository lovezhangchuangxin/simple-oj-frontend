## 快速开始

1. 前置条件：[Node.js](https://nodejs.org/zh-cn/) 和 [pnpm](https://pnpm.io/)。推荐使用 [nvm](https://github.com/nvm-sh/nvm) 来安装 Node.js，本项目使用 Node.js 18.16.0。

2. 安装依赖并启动开发服务器：

```bash
pnpm i
pnpm dev:front
```
